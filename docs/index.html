📖 How to use this page ▼
🎳 Bowl with the number pad Tap 0–9 for how many pins you knock down. X is a strike (all 10 on the first ball); / is a spare (clearing the rest on the second ball). Impossible buttons disable themselves — after an 8, only 0, 1 and / stay lit.
🕹️ Watch the lane Every roll sends the ball down the lane: pins scatter, strikes shake the screen, and rolling a 0 earns you a proper GUTTER. A fresh rack drops in whenever a new frame (or bonus ball) needs one.
🧮 Read the scorecard Each frame shows its rolls on top and the cumulative score below. “…” means the frame is waiting on bonus rolls (a strike needs the next two, a spare the next one). The glowing frame is the one you're bowling.
⌨️ Keyboard shortcuts 0–9 roll, X strike, / spare, ⌫ Backspace undo. On-screen Undo takes back the last roll; Reset starts a new game.
⚡ Demo games The chips below the pad instantly load three famous games — the exercise's example (149), a perfect game (300), and all spares (150) — so you can see a full scorecard without bowling 20 rolls.
🧪 Run the test suite The Run pytest button executes the repo's real test_bowling.py against bowling.py — entirely in your browser. Green means every scoring and validation rule holds. First run installs pytest, so give it a moment.
